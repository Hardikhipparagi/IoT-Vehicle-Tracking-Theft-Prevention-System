# IoT Vehicle Tracking & Theft Prevention System

## Overview

GPS-based vehicle monitoring and theft prevention system using Python simulation and Streamlit dashboard.

## Features

- Vehicle tracking
- GPS simulation
- Google Maps integration
- Geofence monitoring
- Theft detection
- Dashboard visualization
- CSV logging
- PDF reporting

## Tech Stack

- Python
- Pandas
- Streamlit
- Folium
- Geopy
- ReportLab
